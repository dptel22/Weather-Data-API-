# Weather Data API & Web App (Flask)

This project is a Flask-based web application that provides historical weather data through RESTful API endpoints and a simple web interface.

## Overview
The application reads weather station data from text files, processes it using Pandas, and exposes temperature data by station, date, and year via API routes.

## Tech Stack
- Python
- Flask
- Pandas
- HTML (Jinja templates)

## Features
- Display available weather stations on the home page
- Fetch temperature data for a specific station and date
- Retrieve all historical data for a station
- Access yearly weather data through an API endpoint

## Files
- `main.py` – Flask application and API routes
- `about.html` – Static about page
- `tutorial.html` – Web page template
- `tutorial.ipynb` – Development and testing notebook
- `static/` – Static assets (images)
